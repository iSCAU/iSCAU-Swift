iSCAU-Swift
===========

iOS version of "华农宝" and written in ```Swift```.

# Requirements

* Xcode 6.0 and above
* iOS 7.0 and above

# Features

* 接入教务系统，提供课程表获取，成绩查询，考试信息查询，已选选修查询及空余课室查询。
* 接入图书馆系统，支持书籍搜索，已借书籍查询，历史借阅查询。
* 提供校历等更多在校信息帮助。
